# Code of Conduct

This project aims to be friendly, practical, and useful.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Keep discussions focused on the project.
- Welcome new contributors and Android developers of all experience levels.

## Unacceptable Behavior

- Harassment or personal attacks.
- Dismissive or hostile comments.
- Sharing private information without permission.

## Reporting

Open a GitHub issue if you see a problem in project spaces. Maintainers may remove comments, close threads, or block users who make the project unsafe or unproductive.
